# appBack
# 这个框架是由 spring+restlet 组成
# restlet简介：Restlet在术语上参照了Roy Fielding博士论文在讲解REST时采用的术语，如：资源（resource）、表示（representation）、连接器（connector）、组件（component）、媒体类型（media type）、语言（language），等等。这些术语你应该不会陌生。Restlet增加了一些专门的类（如Application、Filter、Finder、Router和Route），用以简化restlets的彼此结合，以及简化把收到的请求（incoming requests）映射为处理它们的资源。
